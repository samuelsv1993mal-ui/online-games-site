# SMQ Games — клиент на телефоне без интернета

Этот режим делает Android-телефон ведущего локальным сервером. Интернет не нужен: друзья подключаются к точке доступа твоего телефона и открывают сайт по локальному адресу.

## Что нужно подготовить один раз, пока есть интернет

1. Установить Termux.
2. Желательно установить Termux:API.
3. Для запуска одной кнопкой можно установить Termux:Widget.
4. Скопировать проект в `~/online-games-site`.
5. В Termux выполнить:

```bash
cd ~/online-games-site
bash scripts/install-phone-client.sh
```

Скрипт установит Node.js, зависимости проекта, QR-утилиту и создаст быстрый запуск `SMQ Games WiFi` для Termux:Widget.

## Как запускать игру без интернета

1. Включи точку доступа на телефоне ведущего.
2. Пусть друзья подключатся к этой Wi-Fi сети.
3. Запусти сервер:

```bash
cd ~/online-games-site
bash scripts/start-phone-client.sh
```

Скрипт покажет адреса, например:

```text
http://192.168.43.1:3000
http://192.168.1.25:3000
```

Друзья открывают этот адрес в браузере. На телефоне ведущего можно открыть:

```text
http://127.0.0.1:3000
```

## Запуск одной кнопкой

После `install-phone-client.sh` можно использовать Termux:Widget:

1. Установи приложение Termux:Widget.
2. На главном экране Android добавь виджет Termux.
3. Выбери ярлык `SMQ Games WiFi`.
4. Перед нажатием включи точку доступа.

## Важно

- Google-вход без интернета может не работать. Используйте вход как гость.
- Termux должен оставаться открытым, иначе Android может остановить сервер.
- Если адрес не открывается, выключи и снова включи точку доступа, затем перезапусти скрипт.
- Некоторые телефоны используют адрес `192.168.43.1`, другие — `192.168.1.x` или `172.20.x.x`. Используй тот адрес, который показывает скрипт.
