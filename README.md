# SMQ Games v15

Точечное обновление после проверки правил игр:

- шахматы: усилена проверка ходов фигур;
- шахматы: запрещены ходы, после которых свой король остаётся под шахом;
- шахматы: добавлена проверка шаха, мата и пата;
- шахматы: добавлена рокировка при корректных условиях;
- шахматы: пешка автоматически превращается в ферзя на последней горизонтали;
- шахматы: доска больше не должна сжиматься на мобильном экране;
- шахматы: фигуры получили стабильный размер и не должны сливаться/накладываться;
- шахматы: 3D-эффект стал мягче, чтобы не ломать читаемость;
- игровые доски шашек и шахмат зафиксированы через aspect-ratio и vmin, чтобы клетки не деформировались;
- обновлён PWA cache до v15.

Проверка синтаксиса выполнена через `node --check` для `server.js` и `public/client.js`.
