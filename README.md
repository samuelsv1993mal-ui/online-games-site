# SMQ Games — Render version

Онлайн-сайт с играми для телефона и компьютера.

## Возможности

- 3 языка интерфейса: русский, español, English.
- Страница входа через Google / гостевой вход.
- Профиль: фото, имя, язык, размер шрифта, светлая/тёмная тема.
- Комнаты онлайн для двух игроков.
- Режим игры с компьютером — SMQ Bot.
- Режим наблюдателя: можно зайти в комнату и смотреть игру без участия.
- Игры: Камень-ножницы-бумага, Крестики-нолики, Кубики, Четыре в ряд.
- Мобильный UI в стиле игрового приложения.

## Запуск локально

```bash
npm install
npm start
```

Открыть:

```text
http://localhost:3000
```

## Render

Build Command:

```bash
npm install
```

Start Command:

```bash
node server.js
```

## Важно про Google Login

Сейчас добавлена страница входа и UX Google-входа для MVP. Реальный Google OAuth требует Google Client ID, Client Secret и callback URL из Google Cloud Console. После получения этих данных можно подключить настоящее OAuth-подтверждение.
