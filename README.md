# SMQ Games v8

Мобильная PWA-версия сайта с онлайн-играми, Firebase Google-входом, профилем, историей, музыкой, fullscreen-ареной и стабильными игровыми досками.

## Что изменено в v8

- Добавлена кнопка **«Скачать как приложение»**.
- Добавлен `manifest.webmanifest` с иконками 192/512.
- Добавлен `service-worker.js`, чтобы сайт устанавливался как PWA на телефон.
- Исправлены игровые поля: клетки больше не сжимаются и не деформируются.
- Крестики-нолики, шашки, четыре в ряд и мемори используют стабильные квадратные сетки.
- Добавлены плавные анимации появления знаков, фишек, карт и победной линии.
- Улучшена адаптация под маленькие экраны, высокий/низкий экран и landscape.
- Улучшена читаемость в светлой и тёмной теме.

## Обновление через Termux

```bash
cd ~
cp /storage/emulated/0/Download/online-games-site-v8.zip ~/
rm -rf ~/online-games-site-v8
unzip -o online-games-site-v8.zip
cp -r ~/online-games-site-v8/* ~/online-games-site/
cd ~/online-games-site
git add .
git commit -m "Stabilize boards and add PWA install"
git push
```

После `git push` Render автоматически задеплоит новую версию.

## Проверка на телефоне

1. Открой `https://online-games-site.onrender.com` в Chrome.
2. Обнови страницу после деплоя.
3. Нажми **«Скачать как приложение»**.
4. Подтверди установку.
5. Открой SMQ Games с иконки на рабочем столе.

Если кнопка установки не появилась сразу, открой меню Chrome `⋮` и выбери **«Добавить на главный экран»**. Иногда браузеру нужен первый успешный запуск service worker, поэтому помогает обновление страницы.
