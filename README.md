# SMQ Games v20

Точечное обновление: игра «Шашки» пересоздана на основе структуры управления шахматами.

## Что изменено

- Шашки больше не используют drag/drop или touch-захват.
- Управление теперь такое же по структуре, как в шахматах:
  1. нажать свою шашку;
  2. увидеть доступные зелёные клетки;
  3. нажать зелёную клетку или кнопку хода снизу.
- Все клетки остаются обычными button-элементами без disabled, чтобы Android/iOS не блокировали касание.
- Клиентская и серверная проверка ходов синхронизированы.
- Обычная шашка ходит вперёд на одну клетку и может бить вперёд/назад.
- Дамка ходит и бьёт по диагонали на любую доступную длину.
- Сохранены fullscreen-доски, режим компьютера и остальные игры.
- PWA cache обновлён до v20.

## Проверка

```bash
node --check public/client.js
node --check server.js
```
