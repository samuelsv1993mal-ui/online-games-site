# SMQ Games v18

Точечное исправление шашек.

## Что исправлено

- Полностью убран drag/drop и любой режим “захвата” шашки.
- Возвращено старое управление: нажать шашку → нажать клетку хода.
- Все тёмные клетки теперь кликабельны на телефоне и компьютере.
- После выбора шашки показываются зелёные разрешённые клетки.
- Под доской добавлены быстрые кнопки доступных ходов, чтобы можно было сделать ход даже если на телефоне неудобно нажать на клетку.
- Клиентская и серверная проверка ходов синхронизированы.
- Обычная шашка ходит вперёд на 1 клетку и может бить назад/вперёд.
- Дамка ходит и бьёт по диагонали в обе стороны на 1–2 клетки, как в старой стабильной логике.
- Обновлён PWA cache до v18.

## Проверка

```bash
node --check public/client.js
node --check server.js
```
