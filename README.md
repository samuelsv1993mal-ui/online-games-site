# SMQ Games v9

Онлайн-сайт игр для Render + Socket.IO + Firebase Auth/Firestore.

## Новое в v9

- Добавлена игра **«Кто я?»**.
- В игре есть **ведущий** — первый игрок, создавший комнату.
- Ведущий может назначить другого ведущего и сам продолжить играть.
- Поддерживается до 10 игроков и наблюдатели.
- Встроена база из 158 русско-испанских вопросов.
- На ходу игрок нажимает **ГО**, запускается анимированная рулетка на 5–15 секунд, потом появляется номер и текст вопроса.
- Игрок может ответить или передать вопрос другому.
- Система очков:
  - ответ на свой вопрос: +2;
  - первая передача: −1;
  - ответ на переданный вопрос: +3;
  - следующая передача: −2;
  - дальше награда и штраф растут по цепочке.
- Ведущий подтверждает или отклоняет ответ.
- Если ответил игрок, которому вопрос был передан, исходный игрок получает право на новый вопрос.
- Добавлены анимации кнопки, появления вопроса и карточек.
- Интерфейс адаптирован под светлую и тёмную тему.
