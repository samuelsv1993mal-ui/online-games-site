# SMQ Games v14

Точечное обновление:

- исправлено управление в шашках: выбор шашки → зелёные клетки → ход, плюс быстрые кнопки доступных ходов;
- добавлены подсказки для новичка в шашках и шахматах;
- улучшена видимость шахматных фигур на мобильном экране;
- шахматная доска получила 3D-наклон и мягкое вращение;
- выбранная игра теперь выделяется более заметной рамкой;
- в профиле добавлена настройка сложности компьютера и игр: beginner / normal / expert;
- сложность передаётся в комнату и влияет на поведение бота в нескольких играх;
- обновлён PWA cache до v14.

Проверка синтаксиса выполнена через `node --check` для `server.js` и `public/client.js`.
