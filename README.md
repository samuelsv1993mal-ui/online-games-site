# Онлайн-сайт с играми для двоих

Готовый MVP-сайт с двумя играми:

- Камень · ножницы · бумага
- Крестики-нолики

Игры работают онлайн через комнаты: один игрок создаёт комнату, второй входит по коду.

## Что внутри

```text
online-games-site/
├── package.json
├── server.js
└── public/
    ├── index.html
    ├── styles.css
    └── client.js
```

## Как запустить

1. Установи Node.js.
2. Открой папку проекта в терминале.
3. Установи зависимости:

```bash
npm install
```

4. Запусти сайт:

```bash
npm start
```

5. Открой в браузере:

```text
http://localhost:3000
```

## Как играть с другом

### На одном Wi‑Fi

1. Запусти сайт у себя.
2. Узнай IP своего компьютера в локальной сети.
3. Друг открывает:

```text
http://ТВОЙ-IP:3000
```

Например:

```text
http://192.168.1.25:3000
```

### Через интернет

Нужно разместить проект на сервере, например Render, Railway, VPS или другой Node.js-хостинг.
После деплоя оба игрока открывают один и тот же адрес сайта.
